# あつ森株価予想

あつまれ どうぶつの森（ACNH）のカブ価推移を予測する、単一ファイル構成のWebアプリケーションです。

## 主な機能
- **高精度な予測**: 解析されたゲーム内アルゴリズムに基づく価格推移と確率の計算
- **グラフ可視化**: Chart.jsを用いた直感的な価格予測範囲（最大値・最小値）の描画
- **自動保存・判定**: 入力データの自動保存（`localStorage`）および、週替わり時の前回パターンの自動算出
- **高騰通知**: 予測最大値が500ベルを超えた際のブラウザプッシュ通知機能

## 使い方
1. 前週の変動パターンと、初回購入かどうかを選択します（不明な場合は自動算出に任せることも可能です）。
2. 日曜日の買値（ウリの販売価格）を入力します。
3. 月曜日から土曜日の午前・午後の売値を入力します。
4. 値を入力するか「予測を実行する」ボタンを押すと、下部に予測されるパターンと推移グラフが表示されます。

## デプロイ（GitHub Pages）
本ツールはバックエンドサーバーを必要とせず、フロントエンド技術のみで完結しています。
このリポジトリをフォークし、GitHub Pagesのソースとして `main` ブランチのルートを指定するだけで、即座に公開・利用が可能です。依存ライブラリはすべてCDN経由で取得されます。
